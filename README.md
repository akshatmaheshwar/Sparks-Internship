# Sparks Bank

A web-based banking management system built during my internship at 
The Sparks Foundation. Developed to learn and understand full-stack 
web development with PHP and MySQL.

## What it does
- Customer management — view and manage bank customers
- Fund transfers between accounts
- Transfer history — full transaction log
- Clean dashboard interface built with Bootstrap

## Tech Stack
PHP · MySQL · HTML · CSS · Bootstrap 4 · JavaScript

## Setup
1. Install XAMPP or any local PHP server
2. Import the database schema into MySQL
3. Update `connection.php` with your database credentials
4. Run the project from your local server (e.g. `localhost/sparks-bank`)
